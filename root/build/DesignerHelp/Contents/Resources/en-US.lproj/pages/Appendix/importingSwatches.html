Importing swatches palettes

Swatches palettes make it easy to store and share color selections. They are managed on the Swatches panel.

Affinity apps can import compatible swatches palette files with an .afpalette file extension.

If swatches palettes are supplied in a ZIP archive, extract the archive's contents before proceeding.

To import swatches palettes:
  1. In the Designer Persona or Pixel Persona, select the Swatches panel.
  2. From the Panel Preferences menu, select Import Palette, and then choose whether to import:
    • As Application Palette—saved within your Affinity app and available to any document.
    • As System Palette—saved to your operating system.
    • As Document Palette—saved within the current document.
  3. Browse to and select the .afpalette file you want to import.
  4. Click Open.

After importing

The new swatches palette is automatically displayed on the Swatches panel.
