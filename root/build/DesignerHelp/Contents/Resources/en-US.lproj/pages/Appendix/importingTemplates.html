Importing templates

Templates are predesigned documents from which new documents can be created and then edited. Like a regular document, each template can contain its own pages/artboards, text styles, assets, swatches palettes and other resources.

Templates are imported via the New Document dialog.

Your Affinity app can import compatible templates with an .aftemplate file extension.

If templates are supplied in a ZIP archive, extract the archive's contents before proceeding.

To import templates:
  1. In Finder, create a folder in which to store templates.
  2. In Explorer, create a folder in which to store templates.
  3. Move or copy your .aftemplate files into the folder.
  4. In your Affinity app, select File>New.
  5. On the New Document dialog, select Templates.
  6. Select Add Folder.
  7. Browse to and select the folder created in step 1.
  8. Click Choose.

After importing

To make a new, unsaved document from a template:
  1. Select File>New.
  2. Select Templates.
  3. Select a folder to see the templates it contains.
  4. Choose a template.
  5. Click Create.
