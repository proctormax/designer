System requirements

The system requirements are as follows:

Minimum:

  • Mac with Apple M1 chip or Intel processor
  • OS X Mavericks 10.9 or later (Supported up to macOS Big Sur 11.0)
  • 2 GB RAM
  • <2.8 GB of available hard-disk space*
  • 1280 x 768 display or better

* additional free space required during installation (cannot install on a volume that uses a case-sensitive file system or on removable flash storage devices).

Metal compute acceleration requires macOS High Sierra 10.13 or later

Registration and content sync feature requires macOS Sierra 10.12 or above.
