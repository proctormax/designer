Importing brushes

Brushes are used with the various brush tools to paint in different styles. They are managed on the Brushes panel.

Your Affinity app can import compatible brushes files with an .afbrushes or .abr file extension.

If brushes are supplied in a ZIP archive, extract the archive's contents before proceeding.

To import brushes:

Do one of the following:

  • In Affinity Designer:
    1. In the Designer Persona or Pixel Persona, select the Brushes panel.
    2. From the Panel Preferences menu, select Import Brushes.
    3. Browse to and select the .afbrushes file you want to import.
    4. Click Open.
  • In a Finder window:
    1. Browse to the .afbrushes file you want to import.
    2. Drag the file over your Affinity app's window.
    3. When the pointer displays a plus sign, drop the file. A dialog will confirm the brushes have been imported.
  • In an Explorer window:
    1. Browse to the .afbrushes file you want to import.
    2. Drag the file over your Affinity app's window.
    3. When the pointer displays a plus sign, drop the file. A dialog will confirm the brushes have been imported.

After importing

A new category containing the brushes is automatically displayed on the Brushes panel.
