Printing artboards

Artboards can be printed individually, all together on one sheet or all on separate sheets.

To print artboards:
  1. From the File menu, select Print.
  2. In the Print dialog, click Show Details.
  3. Ensure Range and Scale is selected from the pop-up menu below the Paper Size setting.
  4. From the Range pop-up menu, select:
    • Entire document: All artboards are printed on a single sheet.
    • Artboards: Prints all artboards, each on a separate sheet.
    • Selection: Prints just the selected artboard.
    • Artboard name: Prints a named artboard.
  5. Adjust other settings in the dialog.
  6. Click Print.
