Design aids and artboards

Unique guide and grid layouts can be established on each artboard to help you position objects. Furthermore, you can activate snapping to dynamically align objects to these guides.

When snapping is active, dynamic guide lines appear when an object is aligned to the edges, corners, or midpoints of the current artboard. You can likewise snap to other document artboards, if they have been established as snapping candidates.

For more information on guides and snapping, see the Guides, Grids and Snapping topics.

To add a ruler guide to an artboard:
  • With an artboard selected, from the View menu, select Guides Manager. Click the Add new guide icon for either horizontal or vertical ruler guides.
To add a column guide to an artboard:
  • With an artboard selected, from the View menu, switch on Show Column Guides then select Guides Manager. The Column Guides section allows you to manage the following:
    • Columns—defines the number of vertical guides that will appear.
    • Rows—defines the number of horizontal guides that will appear.
    • Gutter—defines the size of the gutter.
    • Style—select from Filled or Outline.
    • Color—click the color swatch to select a color for your column guides.
To add a grid to an artboard:
  • With an artboard selected:
    1. From the View menu, select Grid and Axis Manager.
    2. Select Basic mode.
    3. Set the Spacing and Divisions values.
    4. Click Close.

If you choose to display or hide the grid, this will display or hide the grid for all artboards, not just the selected artboard.

To set an artboard as a snapping candidate:

With the Artboard Tool selected, do one of the following:

  • Click on the artboard you wish to make a snapping candidate.
  • Hover over the artboard you wish to make a snapping candidate and wait for the artboard's border to flash purple.
