Paste Style and FX

The Paste Style feature allows you to copy an object's stroke, fill, and layer effect properties and apply them directly to another object. This includes copying between stroke, shape, and text objects.

Paste FX works in the same way as Paste Style but will only apply layer effect properties to the selected object from the clipboard.

To copy and paste style properties:
  1. Select the object with the properties to be copied.
  2. From the Edit menu, select Copy.*
  3. Select the object which will receive the new properties.
  4. From the Edit menu, select Paste Style or Paste FX.

*Alternatively, you can cut (rather than copy) the object. From the Edit menu, select Cut.
