Paste Inside

This feature allows you to paste an object (or selection of objects) inside another object.

To copy and paste inside:
  1. Select one or more objects.
  2. From the Edit menu, select Cut.*
  3. Select the object which will act as the parent object.
  4. From the Edit menu, select Paste Inside.

*Alternatively, you can copy (rather than cut) the object. From the Edit menu, select Copy.
