Range pixel selections

You can create a pixel selection using color or tonal ranges from your image.

If you decide to create a selection based on a color or tonal range, your image is analyzed and all pixels which fall into the chosen range are included in the selection. Pixels which are not in the chosen range are excluded.

Color and tonal range selections are available from the Select menu. These include:

  • From the Color Range sub-menu:
    • Select Reds
    • Select Blues
    • Select Greens
  • From the Tonal Range sub-menu:
    • Select Midtones
    • Select Shadows
    • Select Highlights
