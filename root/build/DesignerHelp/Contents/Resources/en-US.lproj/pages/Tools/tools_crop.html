Vector Crop Tool

The Vector Crop Tool allows you to remove portions of a selected object (or grouped objects) non-destructively and to specific aspect ratios. Images can also be cropped.

To apply a crop:

With the Vector Crop Tool selected, do any of the following:

  • Drag a corner handle to apply a crop from the selected corner.
  • Drag from an edge handle to apply a vertical or horizontal crop from the chosen edge.
To reposition objects within a crop area:
  • With the Vector Crop Tool selected, drag within the crop area.

Modifier keys

When using the Vector Crop Tool, the following modifier keys can be used while applying a crop:

  • The constrains the object's aspect ratio during cropping. You can use this modifier when cropping an image from an edge handle.
  • The temporarily disables snapping.
  • The applies the crop from the center.
  • The allows the crop area to be rotated about its opposite handle.
  • Pressing the right mouse button lets you rotate the crop area about its opposite handle.

These modifiers can be used in combination.
