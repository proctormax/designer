View Tool

The View Tool is used to move the visible portion of your document in the Document view.

The context toolbar for the View Tool also contains the tools to change the zoom level of your document.

If you double-click on the View Tool's icon on the Tools panel, the zoom will be set to 'Fit'.

Tool shortcut : H

Hot key panning:

You can never pan so far that your design disappears out of view, regardless of current zoom level.

Settings

Zoom settings are changed on the context toolbar.

  • Set a zoom level. Select from the pop-up menu.
  • Drag the slider to adjust the zoom level.
  • Set a unit of measurement. Select from the pop-up menu.
