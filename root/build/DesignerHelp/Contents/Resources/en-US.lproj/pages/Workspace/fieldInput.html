Field input

There are many different ways to enter, adjust and convert field units.

Setting field values

Field values can be set using any of the following methods:

  • Input the value directly into the field.
  • Drag slider handles to increase or decrease the field value.
  • Drag left or right across input boxes (only available for certain settings such as Opacity and Transform X, Y, W, H).

Converting field input

Upon entering a field value using measurement units that do not match those used by the field, this will automatically be recalculated and converted to the unit type used in the field. For example, entering a value of 9 cm to a field that uses mm will result in a value of 90 mm.

Fine adjustment of input values (Mouse wheel)

You can use the mouse wheel to change a field's input value.

  • Use the mouse wheel only for a single unit increment.
  • Hold while using the mouse wheel for a 10-unit increment.
  • Hold while using the mouse wheel for a 1/10-unit increment.
  • Hold while using the mouse wheel for a 1/10-unit increment.
