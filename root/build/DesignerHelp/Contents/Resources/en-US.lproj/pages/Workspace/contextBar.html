Context toolbar

The context toolbar always updates to show the options available for the currently selected tool, making it quick and easy to tailor tool-specific settings as you design.

In Normal mode, the context toolbar appears below the Toolbar and immediately above the document view. In Separated mode, if the main toolbar is disabled, the context toolbar floats and can be placed anywhere on your screen.

Each time you select a new tool, the context toolbar shows the commands and options relevant to the selected tool.

In some cases, the commands shown by the context toolbar change depending on selection type and the active Persona. For example, the Node Tool context toolbar will show different options for a shape than it will for a standard vector object made from curves.
