Export

The Export dialog provides a streamlined approach to exporting your document, artboard or current selection, to a common file format by giving you quick access to standard export settings.

For more information on exporting individual artboards, see the Exporting artboards topic.

To export a design as an image:
  1. From the File menu, select Export.
  2. Adjust the settings in the dialog.
  3. (Optional) If one or more layers is selected, the exported file can be trimmed to their boundary by setting Area to Selection Area or Selection Only, which include or exclude all other layers from the output, respectively.
  4. Click Export.
To preview your export:
  1. From the File menu, select Export.
  2. From the dialog, select a raster file format and click Preview.

Modifier keys

When previewing your export, the following modifier keys can be used to zoom in to areas of interest and view the results of different settings applied in the dialog:

  • Hold the while scrolling to zoom in and out of the preview.
  • Hold the while scrolling to scroll left or right.
  • Hold the while scrolling to scroll up or down.
  • Press +1 to zoom to 100%.
To create a custom preset from advanced settings:
  1. From the File menu, select Export.
  2. In the dialog, select More.
  3. Adjust the settings in the dialog.
  4. Click Manage Presets and select Create preset.
  5. Type in the name of the custom preset and click OK. The custom preset will appear in the pop-up menu subsequently.

For more information on the settings available in the More dialog, see the Export Options panel topic.

Settings

Your choice of file format—or preset, which includes a file format—determines which other settings are presented on the dialog.

Commonly used settings are immediately available. Select More to access advanced settings for the chosen format.

For descriptions of the complete range of settings and their availability by format, see Export settings.

Depending on the image format, the export dialog will offer an Estimated File Size for the exported image.
